About Me

Skill yang harus dipelajari untuk menjadi seorang programmer adalah bahasa pemrograman, algoritma dan struktur data,pengembangan web, pengembangan perangkat lunak,basis data, keamanan siber, dan problem solving.

Tiga hal yang harus dikuasai

Python adalah bahasa pemrograman yang mudah dipelajari dan digunakan untuk berbagai aplikasi, termasuk web development, data science, dan automation.

JavaScript (JS) adalah bahasa pemrograman tingkat tinggi yang dinamis dan esensial untuk pengembangan web. Bahasa ini berfungsi membuat halaman website menjadi interaktif dan dinamis, serta digunakan luas di berbagai platform seperti browser (client-side) hingga server (Node.js).

HTML adalah singkatan dari HyperText Markup Language. Ini adalah bahasa markah utama yang digunakan untuk membuat dan menyusun struktur dasar halaman sebuah website.
